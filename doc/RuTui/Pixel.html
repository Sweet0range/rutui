Overview

A Pixel consists of foreground, background and a symbol. Pixels are a central element of RuTui.

Colors can be numbers between 0 and 255

Note: Windows does need a extra software like ANSICON to display more than a few (16) colors.

def initialize foreground = 0, background = nil, symbol = " ", options = {}
	@foreground = foreground
	@background = background
	if symbol.nil?
		@symbol = nil
	else
		@symbol = symbol.to_s[0,1]
	end

	@bold = !!options[:bold]
	@thin = !!options[:thin]
	@italic = !!options[:italic]
	@underline = !!options[:underline]
	@blink = !!options[:blink]
end
